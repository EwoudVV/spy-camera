# spy-camera
A 1980s clock with an esp-32 cam hidden in it. for hack club ispy

i got an old clock from a thrift store, opened it, drilled a hole in the front, and attached an esp-32 cam module, powered from a 9v battery through a buck converter.
the clock itself still works when plugged into an outlet
the clock can still spy when no ac power, because of the 9v battery
the video streams through a wifi hotspot, "Free WiFi", and automatically pops up when joined because of captive thingy.
